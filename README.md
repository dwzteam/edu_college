  > ## 本地快速启动

- 安装[docsify](https://docsify.js.org)

```bash
npm i docsify-cli -g
```

- 启动项目

进入项目根目录下，执行以下命令

```bash
npm run serve
// or
npm run dev
```

当提示`Listening at http://localhost:3000`时，表示服务启动成功。
